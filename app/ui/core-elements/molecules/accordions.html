Star rating

Managers can refine an employee's score using a star and slider combination. A manager can assign a star rating and then select a score for the employee from within the value range for the star (3 stars and a score of 47/100).

Question: How do I turn this on?
Answer: As a suitably authorised administrator, you can enable the Section type setting "Allow manager refinement within item rating range" to allow managers to refine an employee's score using a star and slider combination.

Question: My organisation rates each item by stars, then allows the manager to refine only the section overall score, can this be done?
Answer: Yes, If you display the overall section score for the section, but rate each item, then the refinement will automatically be shown against the section overall score. In addition to the Section type setting "Allow manager refinement within item rating range", you must also enable the Section type setting "Display section overall score" to see this.

Rounding and calibrating performance rating

For organisations that wish to have a more simplified approach to scoring and rating in Performance, PageUp provides the ability to configure performance scores from 0 to 2 decimal places. Where decimal places are used, we also offer organisations the option of rounding calculated, calibrated and overall scores to the nearest whole number.

Question: Where does the rounded rating display, and when?
Answer: On the Next steps > Summary breadcrumb inside a performance review, the Overall rating will be displayed if the review owner has entered a rating for any rateable item in the review (e.g. for an Objective). This is displayed in both the Edit and Rate mode. Also, if the review is in Rate mode and manager calibration is displayed, on the same breadcrumb, the calibrated value is displayed rounded, and the slider is limited to only values with the same number of decimals (e.g. for a rating scale of 1 to 5, if the decimal places displayed is zero (0), the manager can only select a whole value on the slider between 0 and 5 inclusive).
