What is GEL?

PageUp GEL offers visual style, accessible UI components, charts, editorial guidance, UX patterns, and branding released as a HTML & CSS framework, Sketch, and other PDF & AI templates and documented at gel.pageuppeople.com produced by UX team in order to serve 10+ web apps, a few native app and limited branding products and experiences.

Core Components

It provides HTML, CSS, LESS and JavaScript to describe how all of the core interface elements should look and behave. The GUI can also be used as a point of reference for UX and UI designers.
