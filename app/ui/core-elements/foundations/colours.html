Colors should have a semantic value when used in an interface. Stick with these rules:

  • Use green for “Go” actions like “Sign Up,” “Send,” or “Comment”.
  • Use red for “Danger” actions like deleting or removing.
  • Use orange for “Waiting” things like loading, warnings, or non-critical errors.
