Colour Palettes

Colour palettes for our UI design that work harmoniously with our brand colours. It’s also very important to adhere to the accessibility guidelines.
Sketch Print Feedback

    The color palette comes with 9 different weights for each hue, ranging from 100 (lightest) to 900 (darkest.) When using colors start with the 500 weight then go up or down in weight.

    Colors should have a semantic value when used in an interface. Stick with these rules:

    • Use green for “Go” actions like “Sign Up,” “Send,” or “Comment”.
    • Use red for “Danger” actions like deleting or removing.
    • Use orange for “Waiting” things like loading, warnings, or non-critical errors.
