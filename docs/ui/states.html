States

States are visual representations used to communicate the status of a component or interactive element.

Empty states provide users with feedback on the reasons behind that fact and what is possible to be done.

Data doesn’t load instantly, it’s important to show user that their action is in progress.

Spinner

Spinner is for short processes less than 4 seconds. Spinners don’t tell users how long the process will take to load. If you use it for long processes, users will end up wondering if something went wrong with the app.

If a process takes longer than 4 seconds to load, you should use a Progress Bar.

Progress bar

Progress bars make long processes tolerable. Users are more willing to tolerate a long wait time if they see a progress bar.
